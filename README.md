# Online Attendance Count Web Application

This is an online attendance count web application developed using *HTML, CSS, JavaScript* for the frontend and *Node.js with Express.js* and *MongoDB* for the backend. The application allows users to mark attendance and view total attendance records stored in the database.

---

## Features

- Online attendance marking  
- Automatic attendance counting  
- Attendance data stored in MongoDB  
- Responsive and simple user interface  
- RESTful backend API  

---

## Technologies Used

### Frontend
- HTML  
- CSS  
- JavaScript  

### Backend
- Node.js  
- Express.js
